# Pixel Perfect Figma to Web Implementation

## 📌 Overview

This project is a **pixel-perfect conversion of a Figma design into a responsive website**.
The goal of the project is to replicate the original UI design with high accuracy while maintaining clean and structured code.

The landing page includes multiple sections such as features, video, partners, testimonials, pricing, and contact form.

---

## 🚀 Features

* Pixel-perfect UI implementation from Figma
* Responsive design for different screen sizes
* Clean and organized code structure
* Modern SaaS landing page layout
* Multiple UI sections

---

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript
